Initial release of Ghost UI — a behavior-learning layout engine for React.

**`@ghost-ui/core`** — framework-agnostic engine: event observer, optimizer, persistence adapters (memory, localStorage, server, composite), worker optimizer via Comlink.

**`@ghost-ui/react`** — React bindings: `GhostProvider`, `Ghost.Slot`, `Ghost.Button`, `Ghost.Item`, `Ghost.Grid`, `Ghost.Canvas`, `Ghost.Menu`, `Ghost.Tab`, `GhostPrivacyPanel`, `useGhostPrivacy`, `useGhostOrder`, `useGhostScore`, and more.

**`@ghost-ui/devtools`** — in-app score inspector, event timeline, and replay scrubber.

**`@ghost-ui/testing`** — `createTestEngine`, `buildEvents`, `waitForRecompute`, `ghostMatchers`, and `renderWithGhost` for unit and integration tests.
